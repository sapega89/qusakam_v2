Основа:
	Projectile - основной класс для проджектайлов, и не имеет поддержки модулей.
	Projectile_extended - подкласс класса Projectile, добавляет возможность ставить модули,
	а также имеет ID.
	name_hit - название вызываемой функции в узле, которого коснулся проджейктайл
	(в основном это CharacterBody2D). По умолчанию называется "hit".
	Плагин не содержит инструментов вызова проджектайлов, т.к. подразумевается, что вы сами будете
	писать соответствующие для этого функции (не забудьте указать "type" для проджектайла перед его
	спавном в вашей функции)

Модули:
	Модуль - название для ресурсов, которые может использовать Projectile_extended
	Можно использова только в Projectile_extended.
	Atk_change_projectile - основа для изменения атаки проджектайла.
	Speed_change_projectile - основа для изменения скорости проджектайла.
	Move_extended_projectile - основа для изменения передвижения проджектайла.
	Hit_extended_projectile - основа для изменения аргументов, передаваемых в функцию 
	с названием hit.
	Remove_projectile - основа для изменения последствий сталкивания с узлом для проджектайла.

В папке "NZ_projectiles/Scenes" вы сможете найти сцены, в которых сможете посмотреть различные варианты
использования проджектайлов. В сцене "Sandbox" вы можете поэксперементировать с проджектайлами.
Рекомендую посмотреть скрипты, ведь в них указана информация о некоторых переменных и функциях.

Если вы не можете изменить значения переменных в модулях в инспекторе, перейдите на другю сцену, а
затем вернитесь на предыдущую.
