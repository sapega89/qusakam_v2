  ВАЖЛИВО: Виправлення конфлікту аддонів
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА:
Godot виявляє конфлікт між двома аддонами з однаковими класами.

РІШЕННЯ (ОБОВ'ЯЗКОВО):
1. ЗАКРИЙТЕ Godot повністю
2. Перейменуйте папку:
   addons/maaacks_input_remapping 
   → 
   addons/maaacks_input_remapping.disabled
   
   АБО видаліть папку addons/maaacks_input_remapping
3. Відкрийте Godot знову

Детальні інструкції: FIX_ADDDON_CONFLICT.md
