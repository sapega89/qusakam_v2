# Як додати обкладинку гри

## Крок 1: Підготовка зображення

1. Підготуйте зображення обкладинки у форматі PNG або JPG
2. Рекомендовані розміри: **1920x1080** або більше (співвідношення 16:9)
3. Назвіть файл: `game_cover.png` або `game_cover.jpg`

## Крок 2: Розміщення файлу

Помістіть файл обкладинки в одну з цих папок:
- `SampleProject/Resources/Art/game_cover.png`
- `SampleProject/Resources/Art/game_cover.jpg`

## Крок 3: Перевірка

1. Відкрийте проект у Godot
2. Запустіть гру
3. Обкладинка автоматично з'явиться як фон головного меню

## Примітки

- Якщо файл обкладинки не знайдено, буде використано чорний фон
- Обкладинка автоматично масштабується під розмір екрану
- Підтримуються формати PNG та JPG

## Структура папок

```
SampleProject/
└── Resources/
    └── Art/
        ├── game_cover.png  ← Помістіть вашу обкладинку тут
        └── README.md
```
