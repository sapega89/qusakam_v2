# Game Art Assets

## Обкладинка гри (Game Cover)

Щоб додати обкладинку гри:

1. Помістіть файл зображення обкладинки в цю папку з назвою `game_cover.png` або `game_cover.jpg`
2. Рекомендовані розміри: 1920x1080 або більше (16:9)
3. Формати: PNG (з прозорістю) або JPG

Обкладинка буде автоматично використовуватися як фон головного меню.

## Інші арт-ресурси

Ви можете додавати інші арт-ресурси гри в цю папку:
- Спрайти персонажів
- Фонові зображення
- UI елементи
- Іконки
